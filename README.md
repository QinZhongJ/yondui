# yond-ui

## Install
```shell
npm install yond-ui --save
```
## Quick Start
``` javascript
import Vue from 'vue'
import YondUI from 'yond-ui'

Vue.use(YondUI)

// or
import {
  Button
  // ...
} from 'yond-ui'

Vue.component(Button.name, Button)
```
For more information, please refer to [Quick Start](http://marktest.zhlyjg.cn/yond-ui/index.html) in our documentation.
