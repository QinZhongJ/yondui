# y2-ui

## Install
```shell
npm install y2-ui -S
```
## Quick Start
``` javascript
import Vue from 'vue'
import Y2UI from 'y2-ui'

Vue.use(Y2UI)

// or
import {
  Button
  // ...
} from 'y2-ui'

Vue.component(Button.name, Button)
```
For more information, please refer to [Quick Start]() in our documentation.
